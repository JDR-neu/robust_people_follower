Detailed Description

Predicts the target's position if the line of sight to the target is lost. Receives target information from the tracking module. The target's predicted position is calculated using the Constant Turn Rate and Acceleration motion model. Along with the calculation of the predicted position, the Recovery Module calculates a radius in which the target is likely to be located. If a tracked person is inside the radius, it is set as the new target.
